# เนื้อเรื่องย่อ

  “ในเมืองแห่งหนึ่งนามว่า ‘Champignon’ ที่เคยเงียบสงบ ว่ากันว่าขึ้นชื่อเรื่องเห็ดนานาชนิด แต่แล้ววันหนึ่ง ทุกอย่างก็เปลี่ยนไป…”
  
  ไม่มีใครรู้ว่ามันเริ่มขึ้นเมื่อไร จู่ๆ เห็ดประหลาดขนาดยักษ์ก็ผุดขึ้นมาทั่วทั้งเมือง ราวกับมีบางสิ่งเติบโตจากใต้ดิน และพร้อมจะแทรกไซร้ทุกสิ่งบนผิวโลก จากเมืองที่เคยมีชีวิตชีวา กลับกลายเป็นพื้นที่ต้องห้าม เสียงหัวเราะกลายเป็นเสียงกรีดร้อง ทั่วพื้นที่ปกคลุมไปด้วยก๊าซพิษ ตามมาด้วยการย่างกรายเข้ามาของแขกไม่ได้รับเชิญ ชาว Champignon เรียกสิ่งมีชีวิตนี้ว่า “ The Under ” พวกมันดำรงชีวิตอยู่ด้วยการใช้ก๊าซพิษหายใจ ทว่ามนุษย์ไม่เป็นเช่นนั้น หากได้รับก๊าซพิษต่อเนื่องเป็นเวลานานจะทำให้เจ็บป่วยและเสียชีวิตในที่สุด

  คุณ คือหนึ่งในไม่กี่คนที่ยังเหลือรอด ท่ามกลางฝูงมอนสเตอร์ที่ลุกล้ำเข้ามาเพื่อล่าเหยื่อ กับความจริงที่ถูกฝังไว้ใต้ชั้นดินลึก คุณต้องต่อสู้ ไม่ใช่แค่เพื่อมีชีวิตรอด แต่เพื่อหาคำตอบว่า “เห็ดเหล่านี้คืออะไร... และพวกมันต้องการอะไรจากเรา?”
